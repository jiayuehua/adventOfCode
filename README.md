# game24
card 24 point game

usage: ./game.exe [1-13] [1-13] [1-13] [1-13]

example:  
game24.exe 2 4 5 5  
((2*(5+5))+4)  
(4+(2*(5+5)))  
(4+((5+5)*2))  
(((5+5)*2)+4)
